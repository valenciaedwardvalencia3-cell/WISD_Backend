# WISD_Backend

Proyecto backend desarrollado con **Java Spring Boot** y **Maven**.  
Este servicio gestiona usuarios y productos, proporcionando endpoints REST para operaciones CRUD.

## 🚀 Tecnologías utilizadas
- Java 17+
- Spring Boot
- Maven
- JPA / Hibernate
- H2 / MySQL (según configuración)

## 📂 Estructura del proyecto
- `src/main/java/com/wisd/controller` → Controladores REST (Usuario, Producto)
- `src/main/java/com/wisd/model` → Modelos de datos
- `src/main/java/com/wisd/repository` → Repositorios JPA
- `src/main/java/com/wisd/service` → Lógica de negocio
- `src/main/resources` → Configuración (`application.properties`)
- `pom.xml` → Configuración de Maven
